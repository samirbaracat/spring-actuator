# spring actuator

## endpoints

- Listar usuários [GET]:
http://localhost:8080/users

- Buscar usuário [GET]:
http://localhost:8080/users/{id}

- Criar usuário [POST]:
http://localhost:8080/users

- Atualizar usuário [PUT]:
http://localhost:8080/users/{id}

- Alterar status do usuário [GET]:
http://localhost:8080/users/{id}/state

- Excluir usuário [DELETE]:
http://localhost:8080/users/{id}

## monitoramento
http://localhost:8080/actuator
